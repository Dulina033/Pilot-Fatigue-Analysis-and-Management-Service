## API Endpoints

- POST /api/contact (public)
- GET /api/admin/contacts (protected - admin only)
- PATCH /api/admin/contacts/:id/read (protected)
- PATCH /api/admin/contacts/:id/reply (protected)
- DELETE /api/admin/contacts/:id (protected)

## Features

- Contact form with validation
- MongoDB storage
- Email notifications (optional)
- Admin panel for viewing messages
- Status tracking (unread/read/replied)
- IP address and user agent tracking
